String slicing method (str.slice)

The slice() Method slice() extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: the starting index (position), and the ending index (position).

This example slices out a portion of a string from position 7 to position 13:

the sliced string is : 

	
	var str = "Apple, Banana, Kiwi";
	document.getElementById("demo").innerHTML = str.slice(7,13);



The substr() Method

substr() is similar to slice().

The difference is that the second parameter specifies the length of the extracted part.

this example will start slicing in position 7 and then clice 6 characters from that position.

the sliced part of the string is : 

var str = "Apple, Banana, Kiwi";
document.getElementById("demo").innerHTML = str.substr(7,6);

Js Conditions {If , Else}



In JavaScript we have the following conditional statements:

Use if to specify a block of code to be executed, if a specified condition is true

Use else to specify a block of code to be executed, if the same condition is false

Use else if to specify a new condition to test, if the first condition is false

Use switch to specify many alternative blocks of code to be executed.

 
Syntax:

if (condition) {
    block of code to be executed if the condition is true
}



		var x = 5;

		if (x > 4) {
		document.getElementById("new20").innerHTML = "x is greater than 4";

		}else {
			document.getElementById("new20").innerHTML = "x is smaller than 4";
		}
